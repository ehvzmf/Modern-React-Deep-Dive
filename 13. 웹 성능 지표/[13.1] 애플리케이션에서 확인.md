# create-react-app
### `reportWebVitals()`

- 웹에서 성능을 측정하기 위한 함수
    
    
1. 누적 레이아웃 이동(CLS)
2. 최초 입력 지연(FID)
3. 최초 콘텐츠풀 페인트(FCP)
4. 최대 콘텐츠 페인팅(LCP)
5. 첫 바이트까지의 시간(TTFB)

```jsx
// reportWebVitals.ts
import { ReportHandler } from 'web-vitals

const reportWebVitals = (onPerfEntry?: ReportHandler) => {
	if (onPerfEntry && onPerfEntry instanceof Function) {
		import('web-vitals').then(({ getCLS, getFID, getFCP, getLCP, getTTFB }) => {
			getCLS(onPerfEntry)
			getFID(onPerfEntry)
			getFCP(onPerfEntry)
			getLCP(onPerfEntry)
			getTTFB(onPerfEntry)
		})
	}
}

export 
```
